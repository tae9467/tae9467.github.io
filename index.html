About Me

Hey, I am Toula Elwell! A passionate Software Engineering student at RIT, with a flair for creativity and a love for frontend web development. Currently taking an immersion in the fascinating world of Mathematics and pursuing a minor in Web Development, I strive to combine logic and aesthetics in my work.

Throughout my journey, I have embraced exciting challenges, including participating in hackathons where I had the opportunity to develop some truly cool projects. One of my highlights was creating an interactive card game using Python. It was a thrilling experience that showcased my problem-solving skills and ignited my passion for coding.

But my fascination with technology does not stop there! Even back in 8th grade, I indulged in creating my own secret written languages. I quickly discovered that coding shared the same sense of mystery and excitement, making it an incredibly fun endeavor.

What am I seeking now? A frontend web development co-op position! I am fueled by the joy of bringing websites to life. If you are searching for someone who thrives in turning ideas into visually stunning and functional websites, then we are a perfect match.

Social Media Music Website (Work-in-Progress)

Description:

As an aspiring Software Engineer with a passion for web development and music, I am currently working on an exciting personal project called the "Social Media Music Website." This project is heavily inspired by platforms like Instagram and VSCO, with a unique twist: each post plays music based on its mood, sourced from the Spotify API.

Technologies Used:

  • HTML
  • CSS
  • JavaScript
  • React

Tools Used:

  • Figma
  • VSCode
  • Github
  • Coolors

Key Features (Planned):

  • User-friendly interface with an Instagram/VSCO aesthetic.
  • Seamless integration with the Spotify API for a mood-based music selection.
  • Creating and sharing posts with personalized music choices.
  • React components for smooth and efficient rendering.
  • Responsive design for optimal viewing on various devices.

Project Progress

As of now, the "Social Media Music Website" project is in its development phase. I am actively working on the implementation of the core functionalities and refining the user interface using Figma. The website's front-end is being built with React, ensuring a dynamic and engaging user experience.

Insights:

This project allows me to combine my passion for music and web development. The ongoing process of working with APIs, designing visually appealing layouts, and using React to create interactive components has been an enriching learning experience. I'm excited to see this project come to life and make a positive impact on users who are equally passionate about music and social connections.
